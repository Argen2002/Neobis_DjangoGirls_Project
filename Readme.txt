Neobis_DjangoGirls_Project

Описание проекта
Проект блог где было реализовано-  добавление, удаление, изменение своего поста.


Установка
Склонируйте репозиторий в вашу папку где будет находится сам проект.

Открыть с помощью Pycharm или других IDE

Создайте виртуальное окружение используя команду
python3 -m venv venv
 
Активируйте виртуальное окружение
source venv/bin/activate 

Установите зависимости
pip install -r requirements.txt 


Сделайте миграции

  ./manage.py migrate
Запусти свой проект
./manage.py runserver 
или
 python3 manage.py runserver
